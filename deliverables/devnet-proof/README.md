# EarnScout Devnet Proof

This folder is reserved for the public proof generated by the no-Docker settlement script.

Run:

```sh
cd examples/txodds
npm install
SOLANA_RPC_URL=https://docs-demo.solana-devnet.quiknode.pro/ npm run earnscout:settle
```

Prerequisite: the disposable buyer wallet generated by `npm run setup` needs at least `0.005` devnet SOL.

When funded, the script writes:

- `earnscout-settlement.json`
- `earnscout-settlement.md`

Those files include only public proof data: buyer/seller addresses, escrow PDA, delivery hash, deposit
transaction, and release transaction. They do not include private keys.
